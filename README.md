# Gmail Cleanup AI

AI-powered Gmail cleanup assistant built with:

- FastAPI
- PostgreSQL
- pgvector
- Redis
- React (planned)

## Features (WIP)

- Gmail email ingestion
- Email storage and indexing
- Rule-based classification
- AI-powered recommendations
- Vector similarity search (planned)

## Local Development

Run:

docker compose up --build

API:

http://localhost:8000

Health check:

/health
